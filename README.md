# @amlplugins/mistral-batch

Native `@mistralai/mistralai` SDK re-exported under the `@amlplugins` namespace.

> Mistral Batch API — async batch processing at 50% discount with JSONL request/response files.

## Install

```bash
npm install @amlplugins/mistral-batch
```

## Use

```ts
import { /* named exports from @mistralai/mistralai */ } from "@amlplugins/mistral-batch";
// or
import sdk from "@amlplugins/mistral-batch";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
